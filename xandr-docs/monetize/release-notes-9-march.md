# Seller Monitoring Workflow (SMW) UI modernization now generally available

**Release date:** March 9, 2026  
**Product:** Microsoft Monetize  
**Area:** Seller Monitoring Workflow (SMW)  
**Status:** General Availability (GA)

## Overview

Microsoft Monetize has completed the modernization of the **Seller Monitoring Workflow (SMW)** by migrating the user interface from **anxreact** to **Fluent UI**. This update improves accessibility, aligns Monetize with Microsoft-wide UI standards, and provides a foundation for future product enhancements.

This modernization delivers the following benefits:

- Ensures compliance with Microsoft accessibility standards (WCAG)
- Consolidates Monetize on a single, supported UI framework
- Reduces technical debt from legacy components
- Enables faster and more consistent UI improvements going forward

The Fluent UI experience is now fully available across the **Advertisers**, **Insertion Orders**, and **Line Items** screens.

## What’s new

The updated SMW experience introduces a modern Fluent UI interface across all workflow screens.

Key improvements include:

- Updated **Advertisers**, **Insertion Orders**, and **Line Items** interfaces built on Fluent UI
- Improved accessibility, including:
  - Enhanced keyboard navigation
  - Improved screen reader support
- Modernized table layouts with improved hierarchy and spacing
- Consistent interaction patterns aligned with other Microsoft applications

These improvements provide a more consistent and accessible experience across Microsoft Monetize.

---

## What remains unchanged

The UI update does not change core platform functionality. The following remain unchanged:

- Seller Monitoring Workflow functionality and workflows
- Advertiser, Insertion Order, and Line Item management behavior
- Data integrity, permissions, and reporting logic

Users can continue managing their inventory and delivery without disruption to existing workflows.

---

## Known limitations

At initial GA, the following limitations may apply:

**Quick filters**  
Some quick filtering capabilities may be temporarily unavailable. Support for these filters is planned for a future update.

**Saved views**  
Saved views are not available at GA and are planned as a fast-follow enhancement.

These limitations do not affect core SMW workflows.
