# Enable Cookie Set

<div class="body">

<div class="note">

<span class="notetitle">**Note:**</span> This function is scheduled to be deprecated.
</div>

This function is used to set cookies via link redirects. It only
supports Safari browser (for now). If you use it for other browsers,
nothing will happen.

``` pre
enableCookieSet()
```

<div class="section">

## Example

``` pre
apntag.enableCookieSet()
```
