# Deals Screen in Microsoft Monetize

## Overview

The **Deals screen** in Microsoft Monetize provides a centralized workspace for managing programmatic deals between sellers and buyers. It allows Ad Operations teams and programmatic sales teams to create, monitor, and optimize deals while tracking delivery and performance metrics.

Using the Deals screen, you can:

- View all deals available in your network
- Monitor deal delivery and performance
- Create and edit deals
- Troubleshoot deal delivery issues
- Filter and export deal data

The Deals screen acts as the primary interface for managing deals throughout their lifecycle.
