# Navigate to the Create New Line Item Screen



You can create new augmented line items from the
Create New Line Item screen.



1.  Click the value that displays under the
    Line Items column for the
    corresponding advertiser.
2.  Click + New Line
    Item.
    

    The Create New Line Item
    screen displays.

    
3.  Select the
    Augmented option.
4.  Search and select an advertiser from the
    Advertiser text field.
5.  Search and select an insertion order from the
    Insertion Order text field.
6.  Click
    Next.
