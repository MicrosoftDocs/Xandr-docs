# Deals Screen in Microsoft Monetize

## Overview

The **Deals screen** in Microsoft Monetize provides a centralized workspace for managing programmatic deals between sellers and buyers. It allows Ad Operations teams and programmatic sales teams to create, monitor, and optimize deals while tracking delivery and performance metrics.

Using the Deals screen, you can:

- View all deals available in your network
- Monitor deal delivery and performance
- Create and edit deals
- Troubleshoot deal delivery issues
- Filter and export deal data

The Deals screen acts as the primary interface for managing deals throughout their lifecycle.

## Where the Deals Screen Fits in the Deal Lifecycle

The Deals screen supports multiple stages of the deal lifecycle.

Typical deal workflow:

Negotiate Deal → Create Deal → Monitor Delivery → Optimize → Analyze Performance


You can use the Deals screen to:

- Monitor deal health and delivery
- Review buyer participation
- Adjust deal configuration
- Diagnose issues affecting deal performance

---

## Access the Deals Screen

To access the Deals screen:

1. Log in to **Microsoft Monetize**.
2. Navigate to **Monetize → Deals**.
3. The Deals screen displays a list of all deals configured in your network.

Each row in the table represents a deal and provides key details such as status, buyer information, pricing, and delivery metrics.

---

## Key Components of the Deals Screen

The Deals screen contains several components designed to help you manage deals efficiently.

### Deals Grid

The **Deals grid** displays all deals in a table format.

Typical columns include:

- Deal ID
- Deal name
- Buyer
- Deal type
- Status
- Pricing model
- Delivery metrics

You can sort columns and adjust views to focus on the most relevant information.

---

### Filters and Search

Use filters and search tools to quickly locate deals.

Common filters include:

- Deal status
- Buyer
- Deal type
- Inventory targeting

Filtering helps identify specific deals or groups of deals for monitoring and troubleshooting.

---

### Deal Details Panel

Selecting a deal opens a details panel where you can view additional information such as:

- Inventory targeting
- Pricing configuration
- Buyer seat information
- Delivery metrics
- Deal eligibility status

From this panel, you can review and update deal settings.

---

## Supported Deal Types

Microsoft Monetize supports several types of deals.

| Deal Type | Description |
|-----------|-------------|
| Private Marketplace (PMP) | Private auction between sellers and selected buyers |
| Preferred Deal | Fixed-price agreement with priority access to inventory |
| Programmatic Guaranteed | Guaranteed delivery at a fixed price and volume |

Different deal types provide different levels of control over pricing, access, and delivery.

---

## Monitor Deal Performance

The Deals screen provides performance metrics that help evaluate how deals are performing.

Common metrics include:

| Metric | Description |
|------|-------------|
| Impressions | Total impressions delivered through the deal |
| Revenue | Revenue generated by the deal |
| Win Rate | Percentage of bids won |
| Fill Rate | Percentage of ad requests filled |
| Bid Rate | Percentage of requests that receive bids |

Monitoring these metrics helps identify underperforming deals and opportunities for optimization.

---

## Troubleshoot Deal Delivery Issues

If a deal is not delivering as expected, the Deals screen can help identify potential issues.

### Common Issues

| Issue | Possible Cause | Resolution |
|------|---------------|-----------|
| No bids received | Buyer not targeting deal ID | Verify buyer targeting configuration |
| Low delivery | Inventory targeting too restrictive | Review inventory targeting settings |
| Deal not eligible | Floor price set too high | Adjust pricing or renegotiate with buyer |
| Low fill rate | Limited buyer demand | Expand buyer participation |

Use filters and performance metrics to isolate problematic deals and investigate delivery issues.

## Best Practices for Managing Deals

Follow these best practices to improve deal management and performance:

- Use **clear naming conventions** for deals
- Monitor delivery metrics regularly
- Track buyer participation and engagement
- Review deal targeting periodically
- Archive inactive or expired deals

Maintaining organized deal management practices helps improve operational efficiency and reporting accuracy.
