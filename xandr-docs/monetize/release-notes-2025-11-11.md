# New Deals Screen in Microsoft Monetize

**Release date:** November 11, 2025  
**Product:** Microsoft Monetize – Deals screen  

## Overview
The new **Deals Screen** in Microsoft Monetize provides a single, streamlined workspace to create, edit, monitor, and troubleshoot every deal type.  
This update delivers a faster, more intuitive experience that reduces manual effort, increases visibility, and keeps all key deal information within easy reach.

## What’s new
- **Unified workspace:** View all deals together or switch between dedicated tabs for each deal type.  
- **Time-range control:** Choose the reporting window that best matches your analysis.  
- **Customizable grid with saved views:** Adjust columns, sorts, and filters to fit your workflow, and bookmark URLs for easy reuse or sharing.  
- **Multilevel sorting:** Organize deals by multiple criteria to surface the most important ones first.  
- **Advanced search and filters:** Find deals quickly, including by multiple IDs in a single search.  
- **Export (download):** Export deal data with metrics for offline analysis or reporting.  
- **Flexible deal creation:** Create new deals from scratch or use existing ones as templates to accelerate setup.  
- **Contextual side pane:** Access settings, health, performance, troubleshooting, and history without leaving the grid.  
- **Bulk actions:** Activate, deactivate, archive, delete, or run reports across multiple deals at once.  
- **Archived-deals view:** Review historical deals separately to keep your active workspace focused.  

## Supported deal types
- **Standard deals**  
- **Guaranteed deals**  
- **Code-mapped deals** (for OpenRTB integrations where a code in the bid request maps to a Microsoft Monetize deal ID)  

## Why it matters
This release unifies and simplifies deal management in Microsoft Monetize.  
By combining setup, performance monitoring, and troubleshooting in a single interface, you can work faster, maintain cleaner workflows, and make data-driven decisions with confidence.

## Get started
Sign in to **Microsoft Monetize → Deals** to explore the new experience.  
For step-by-step guidance, see [Deals Screen in Microsoft Learn](deals-screen-in-microsoft-monetize.md).
