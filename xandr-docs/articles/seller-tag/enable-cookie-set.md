# Enable Cookie Set





Note:

This function is scheduled to be deprecated.



This function is used to set cookies via link redirects. It only
supports Safari browser (for now). If you use it for other browsers,
nothing will happen.

``` pre
enableCookieSet()
```



## Example

``` pre
apntag.enableCookieSet()
```
