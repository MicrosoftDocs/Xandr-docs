# Check Order Performance



Use the Order Performance widget to quickly view how an order or order
line is performing.

1.  On the Order Dashboard page, view the Order Performance widget.
2.  From the drop-down menu, select whether to display performance By
    Order or By Order Line.
3.  (Optional) Adjust the date range by clicking the Last x
    Day(s) filter, specifying a new standard or custom date range, and
    clicking Update. 
4.  Mouse over the order or order line to view a summary of its details
    in a tooltip.
5.  Click a specific order or order line to view the Order Visibility
    or Order Line Visibility pages.
6.  (Optional) On the Order Line Visibility page,
    click What should I do? to
    launch Scenario Manager.
