# Microsoft Curate Home Page

The Microsoft Curate Home page is the default landing experience for all Microsoft Curate customers. It provides a centralized, action-oriented view of business performance, key performance indicators (KPIs), workflow shortcuts, and actionable insights. The page is designed to help stakeholders quickly understand performance trends, stay informed, and take effective action toward achieving daily business goals. 

The Home page consolidates key modules, including Seller Monitoring Workflow (SMW) line items and insights, and enables users to efficiently navigate tasks while maintaining visibility into revenue and margin performance. 
 

## Navigate the home page

When you sign in to Microsoft Curate, the Home page is displayed by default. It consists of two primary sections: 
- Performance changes 
- For You 

Each section is designed to provide clarity, prioritization, and direct pathways to action. 
 

### Performance changes 

The Performance changes section provides a high-level overview of key Curate business metrics over a selected comparison window. It indicates whether core metrics are trending upward or downward over the selected time period. 
- Metrics shown in this section include: 
    - Curator revenue 
    - Curator margin (in applicable views) 
    - Seller or deal revenue 
    - Directional indicators (increase or decrease) 
- This section provides a quick pulse check when landing on Curate and helps answer whether the business is generally trending up or down. Based on the direction of change, users can decide whether to drill into reports or specific deals for deeper analysis. To adjust the timeframe for the performance data, use the date range selector located at the top-right of the Performance section. The available options are: 
    - Last 7 days 
    - Last 28 days 
    - Yesterday 
- Within this section, you can see changes in Curator Revenue, Curator Margin (In some views) Seller / Deal revenue, Directional indicators (increase vs decrease) over the selected date range. Each card is fully interactive and you can hover over any graph to view the delta in revenue changes. 

### For You 
The For You section provides a curated set of actionable data, workflow shortcuts and personalized insights designed to streamline your daily workflow. This section includes the following subsections: 

#### Quick create 

The Quick create panel provides fast access to commonly used Curate workflows like creating reports or plans. It functions as a shortcut card that enables users to initiate key actions without navigating through menus. This enables you to perform one of the following actions: 

- Create a new report 
- Create a new plan 
- Create a Curated deal 

Selecting a Create option on either one of the menu options, redirects you to the corresponding creation page, where you can begin configuring the selected object. For additional guidance on these workflows, refer to: 
- [Reporting Guide](reporting-guide.md)
- [Curate Planner](planner.md)
- [Selling Curated Deals](selling-curated-deals.md)

#### Saved Reports 

The Saved Reports panel displays the five most recently run saved reports associated with your user profile. You can take the following actions: 

- Select Run to quickly and instantly execute your saved report. 
- Select Edit to open the report in a separate window, modify report parameters and then select Run Report to execute the report. 
- If no saved reports are available, click the Report button to quickly access reporting tools and create a new custom report. 

#### Top spend increases 

The Top spend increases section highlights deals where spend has increased significantly over a selected timeframe. This section surfaces unexpected growth or new demand and provides visibility into meaningful performance shifts. This section helps curators: 

- Identify deals that recently started spending 
- Detect deals scaling faster than expected 
- Validate successful performance 
- Confirm supply and demand health 
- Ensure margins remain acceptable 
- Proactively communicate performance wins 

Selecting any row opens the corresponding Seller Monitoring Workflow (SMW) Line Items grid. From there, you can review line item settings and make updates to ensure they continue serving as expected. To adjust the timeframe for the top spend increases data, use the date range selector located at the top-right of the top send increases section. The available options are: 

- Last 7 days 
- Last 28 days 
- Yesterday 
 

#### Top spend decreases  

The Top spend decreases section highlights deals where spend has declined sharply over a selected timeframe. These declines may indicate delivery issues, demand fluctuations, or revenue risk. This section helps answer: 

- “Which deals just stopped or nearly stopped spending?” 
- “What should I investigate immediately?” 

It enables proactive troubleshooting and revenue protection. Selecting a row opens the corresponding Seller Monitoring Workflow (SMW) Line Items grid, where you can inspect either, Revenue, Delivery, Bid rate, or Rejection issues. 
 
To adjust the timeframe for the top spend increases data, use the date range selector located at the top-right of the top send increases section. The available options are: 

- Last 7 days 
- Last 28 days 
- Yesterday


#### Recently Modified 

The Recently Modified section displays the five most recently modified objects in your environment. This feature is designed to help you quickly return to recent tasks or items that you were working on. You can select any object on the card to view or review its corresponding details and update them to ensure they start serving as expected. 
 

#### Notifications 

The Notifications section provides timely updates about platform events, including: 

- System outages and maintenance updates. 
- Product release notes and new feature announcements. 
- When system notifications are available, click them to be redirected to the relevant Status or Release Notes page on Microsoft Learn for additional insights and details. 


## Edit the home page 
The Edit mode on the Curate Home page allows users to customize the layout of the For You section. This mode enables card configuration and layout personalization. Within Edit mode, you can perform the following actions: 
- **Reorder cards:** Drag and rearrange cards within the For You section to align with your workflow priorities. 
- **Add a new card:** Select Add to open the Card Library, which contains predefined and configurable cards.  Once added, the card displays performance insights for the selected line item. Selecting a row within the card opens the corresponding Seller Monitoring Workflow (SMW) Line Items grid, where you can review settings and make updates as needed. Cards are organized into the following tabs: 
    - **System tab:** The System tab contains predefined cards created by Microsoft for users to add to their Home page. For example, selecting the Top rejected bid rates card allows you to see which deals are most frequently blocked. 
    - **Line Item tab:** The Line Item tab allows you to create a custom line item card. To add a line item card: 
        - Search for a line item. 
        - Select the desired line item. 
        - Select **Add** to place it on your home page. 
    - **Delete cards:** Hover over any card in the For You section when not in edit mode: 
        - Click the three-dot menu and select remove card.  
        - The card is then placed in the Card Library and can be re-added when in Edit mode.
