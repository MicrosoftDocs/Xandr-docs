# Basic Buy-side Setup Procedures



This page provides a list of procedures needed to get your buying
objects quickly setup on the Xandr platform. You
can use these to set up media buying on behalf of your company or, if
you're an agency, your advertisers.

<div id="ID-000015f4__section-51956117-1a8f-4d1f-a66d-1e210fbd1cdf"
>

The sequence of procedures you follow depends on whether you plan to use
**Standard** line items or
<a href="augmented-line-items-ali.html" class="xref">Augmented Line
Items (ALI)</a>. Note that ALIs cannot be used with legacy insertion
orders.



Important: **Determine which type of
line item you are currently using.**

currently using.
