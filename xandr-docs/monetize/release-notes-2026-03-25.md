# Inventory Manager UI modernization now generally available

**Release date:** March 25, 2026
**Product:** Microsoft Monetize
**Area:** Inventory Manager
**Status:** General Availability (GA)

## Overview

Microsoft Monetize has completed the modernization of **Inventory Manager**, migrating the user interface to **Fluent UI**. This update improves accessibility, aligns Monetize with Microsoft UI standards, and provides a foundation for future enhancements.

The Fluent UI experience is now fully available in the **Inventory Manager** screen under the **Publishers** tab.

This update provides the following benefits:

* Improves compliance with Microsoft accessibility standards (WCAG)
* Aligns Monetize with Microsoft Fluent UI design standards
* Reduces reliance on legacy UI components
* Enables faster and more consistent UI improvements in future releases

---

## What's new

The updated experience introduces a modern Fluent UI interface with several usability and accessibility improvements.

Key improvements include:

* **Modernized Inventory Manager interface** built with Fluent UI, providing consistent interaction patterns aligned with other Microsoft applications
* **Tree view** for browsing inventory hierarchy, allowing users to expand placement groups and view their associated placements
* **Improved accessibility**, including:

  * Enhanced keyboard navigation
  * Improved screen reader support
  * Updated table layouts with improved hierarchy and spacing

These updates provide a more consistent and accessible experience across Microsoft Monetize.

---

## What remains unchanged

This update does **not** change core platform functionality. The following remain unchanged:

* Inventory Manager create and edit workflows
* Data integrity and permissions
* Reporting logic and metrics

Users can continue managing inventory and delivery without disruption to existing workflows.
